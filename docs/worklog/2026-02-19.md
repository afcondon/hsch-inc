## Decisions Made

- **`clean-test` depends on `stop`**: Always kill existing services before cloning, so the test clone can bind to 3000/3001 without conflict
- **Sleep in `stop` target**: Ports need time to fully release after process kill — 1s sleep prevents the EADDRINUSE race we hit twice
- **Test dir is `/tmp/minard-test`**: Ephemeral, outside the workspace, easily cleaned up. Hardcoded rather than parameterized since it's always the same workflow
