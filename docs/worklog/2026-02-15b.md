## Decisions Made

- **Render-then-measure over heuristics**: More reliable for variable-width HTML content
- **Clear before siglet fallback**: HATS rerender does tree diffing (GUP), so replacing a sigil with a structurally different siglet requires clearing the DOM first
- **`--outfile` path for spago bundle**: Must be relative to package directory (`public/bundle.js`), not workspace root (`frontend/public/bundle.js`)
